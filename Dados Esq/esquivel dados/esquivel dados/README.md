# Simulador de Lanzamiento de Dados en C#

Este proyecto es un simulador de lanzamiento de dados en C#. 
A continuación, Te detallo sus características y cómo funciona:

## Descripción

Este programa en C# simula el lanzamiento de dos dados virtuales y muestra los resultados en la consola.
Los usuarios pueden lanzar los dados presionando la tecla Enter y salir del programa en cualquier momento presionando la tecla Esc.

## Funcionalidades

- **Lanzamiento de Dados**: El programa genera números aleatorios entre 1 y 6 para simular el lanzamiento de dos dados. Los resultados se muestran en la consola.

- **Interfaz Amigable**: La consola muestra un mensaje de bienvenida y las instrucciones para lanzar los dados o salir del programa.

- **Salida Controlada**: El programa permite a los usuarios salir en cualquier momento presionando la tecla Esc.
